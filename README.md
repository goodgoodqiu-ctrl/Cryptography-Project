# 同态加密与加密通信演示平台

本项目是为密码学课程设计的大作业演示系统。采用 Python GUI 框架构建，集成了“基于 Paillier 算法的隐私计算投票系统”与“基于 RSA+AES 的混合加密安全通信系统”。

# 核心功能模块

模块一：IoT 设备特征同态加密计算 
技术方法: Paillier 算法
应用场景: 在 IoT 设备身份识别网络中，边缘网关需要收集各个传感器的特征数据进行聚合或加权运算。
原理: 利用同态加密特性，传感器在本地将特征值进行加密，网关可以在绝对不解密的情况下，直接对密文执行加法、减法和标量乘法。例如执行 $E(Feature_A) + E(Feature_B)$，并生成全新的密文。云端获取该密文后进行私钥解密，所得明文完美对应数学运算的结果。

模块二：混合加密安全通信 
技术方法: RSA (非对称加密) + AES-EAX (对称加密/认证加密)
原理: 结合了对称加密的高效与非对称加密的密钥分发安全性。 每次通信随机生成一次性 AES 会话密钥加密海量数据，再使用接收方的 RSA 公钥加密该 AES 密钥。模拟了 HTTPS/TLS 底层的安全握手与传输逻辑。
